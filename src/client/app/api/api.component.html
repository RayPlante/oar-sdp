This NIST Science Data Portal page hosts access to a web based public API, Application Programming Interface. This API provides our user community the capability to dynamically query the underlying data resource catalog, a collection of key NIST research dataset descriptions (metadata), with network enabled computer programs. The portal search API leverages integration of standard metadata representations and indexing of domain services to enable discovery and data mining of the NIST research public datasets. In addition to the main portal API, this page includes a listing of individual key resource APIs hosted at NIST serving scientific domain expertise and access to the underlying datasets.
